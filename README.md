# Atividade Prática 8 — Formas Abstratas em Java

Projeto desenvolvido no IntelliJ IDEA.
Contém classes abstratas e concretas de formas geométricas (2D e 3D).

## Arquivos principais
- `FormaGeometrica.java`: classe abstrata base
- `FormaBidimensional.java` e `FormaTridimensional.java`: subclasses abstratas
- `Quadrado.java`, `Retangulo.java`, `Circulo.java`, `Cubo.java`: classes concretas
- `TesteFormas.java`: classe de teste (contém o método `main`)
